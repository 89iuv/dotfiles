# Dotfiles

## Distro
Tested on Ubuntu 22.04 running on Windows 11 wls

## Install
1. Run commands in home folder:
```
git clone --recursive git@github.com:89iuv/dotfiles.git .dotfiles
cd .dotfiles && ./install.sh
```
2. Restart shell

## Update
``` 
git pull
git submodule update --recursive --remote 
```
